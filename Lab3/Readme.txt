This directory contains following  files:

1. 3array.cpp : This is a C++ program which finds the common elements from three arrays in time O(n).
2. printf.cpp : This is a C++ program which prints text on screen without any printf statement.
3. printself.cpp :  Thsis is a C+ program which prints the same program.
4. input.txt : A text file which is the input for 3array.cpp.


The scripts for compiling and running the above programs are named same as the programs. So to run a program run the script with same name as program name.
