This directory contains following  files:

1. sort.cpp : This is a C++ program which provides four sorting algorithms to sort a given sequence which are namely : Selection Sort, Insertion Sort , Quick Sort and Merge Sort.The program asks the user to select one of the following algorithms to sort the sequence and stores back the sorted sequence in a file named output.txt.



The scripts for compiling and running the above programs are named same as the programs. So to run a program run the script with same name as program name.
